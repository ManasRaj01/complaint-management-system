# 📌 Complaint Management System

A full-stack MERN (MongoDB, Express, React, Node.js) application to manage complaints efficiently. This system allows users to register complaints, track their status, and provide feedback.

---

## 🚀 Features

- 👤 User Registration & Login (Authentication)
- 📝 Submit Complaints
- 📊 Track Complaint Status
- 🧑‍💼 Admin Dashboard
- 💬 Feedback System
- 🔒 Secure Backend (JWT Authentication)

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/ManasRaj01/complaint-management-system.git

cd complaint-management-system

---

### 2️⃣ Setup Backend
cd backend
npm install

Create a `.env` file and add:
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key


Run backend:
npm start


---

### 3️⃣ Setup Frontend
cd frontend
npm install
npm start

---

## 🌐 Usage

- Open browser and go to: `http://localhost:3000`
- Register/Login
- Submit complaints
- Track status
